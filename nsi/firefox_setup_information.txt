=====Firefox Default Setup Information=====
The following include information on how I setup a new firefox installation.

[[nsi:Configure NTLM Passthrough]]

Install the following addons:

  *  Web Development / Security:
    * [[https://addons.mozilla.org/en-US/firefox/downloads/latest/60/addon-60-latest.xpi?src=developers Web| Developer Toolbar]]
    * [[https://addons.mozilla.org/en-US/firefox/downloads/latest/59/addon-59-latest.xpi?src=addondetail| User Agent Switcher]]
    * [[https://addons.mozilla.org/en-US/firefox/downloads/latest/1843/addon-1843-latest.xpi?src=addondetail| Firebug]]
    * [[https://addons.mozilla.org/en-US/firefox/downloads/latest/966/addon-966-latest.xpi?src=category| Tamper Data]]
    * [[https://lastpass.com/lp_win.xpi| Last Pass]]

  *  Utilities:
    * [[https://addons.mozilla.org/en-US/firefox/downloads/latest/1865/addon-1865-latest.xpi?src=addondetail| Adblock Plus]]
    * [[https://addons.mozilla.org/en-US/firefox/downloads/latest/201/addon-201-latest.xpi?src=addondetail| DownThemAll]]
    * [[https://addons.mozilla.org/en-US/firefox/downloads/latest/26/addon-26-latest.xpi?src=addondetail| Download Status bar]]
    * [[https://addons.mozilla.org/en-US/firefox/downloads/latest/92382/platform:5/addon-92382-latest.xpi| IETab 2]]
    * [[https://addons.mozilla.org/en-US/firefox/downloads/latest/433/addon-433-latest.xpi?src=addondetail| Flash Block]]
    * [[https://addons.mozilla.org/en-US/firefox/downloads/latest/2464/addon-2464-latest.xpi?src=addondetail| Foxy Proxy]]
    * [[https://addons.mozilla.org/en-US/firefox/downloads/latest/6366/addon-6366-latest.xpi?src=addondetail| Fire Gestures]]
    * [[https://addons.mozilla.org/en-US/firefox/downloads/latest/4287/addon-4287-latest.xpi?src=addondetail| FoxSplitter]]
    * [[https://addons.mozilla.org/en-US/firefox/downloads/latest/5447/addon-5447-latest.xpi?src=addondetail|TabKit]]
    * [[https://addons.mozilla.org/en-US/firefox/downloads/latest/4550/platform:2/addon-4550-latest.xpi?src=addondetail|Compact Menu 2]]
