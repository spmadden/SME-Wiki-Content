=====Notable RFCs from the IETF=====
  * RFC 1149 - [[http://tools.ietf.org/html/rfc1149|IP Over Avian Carriers]]
    * Extended by RFC 2549 - [[http://tools.ietf.org/html/rfc2549|IPoAC With QoS]]
    * Extended by RFC 6214 - [[http://tools.ietf.org/html/rfc6214|IPoAC using IPv6]]
  * RFC 1925 - [[http://tools.ietf.org/html/rfc1925|The Twelve Truths of Networking]]
  * RFC 2100 - [[http://tools.ietf.org/html/rfc2100|The Naming of Hosts]]
  * RFC 2324 - [[http://tools.ietf.org/html/rfc2324|Hyper Text Coffee Pot Control Protocol (HTCPCP/1.0)]]
  * RFC 2440 - [[http://tools.ietf.org/html/rfc2440|The OpenPGP Message Format]]
    * Obsoleted by RFC 4880 - [[http://tools.ietf.org/html/rfc4880|The OpenPGP Message Format]]
  * RFC 2119 - [[http://www.ietf.org/rfc/rfc2119.txt|Key words for use in RFCs to Indicate Requirement Levels]]
  * RFC 2828 - [[http://tools.ietf.org/html/rfc2828|Internet Security Glossary]]
  * RFC 2898 - [[http://tools.ietf.org/html/rfc2898|PKCS#5 Password Based Crypto Specification]]
  * RFC 5869 - [[http://tools.ietf.org/html/rfc5869|HMAC-based Extract-and-Expand Key Derivation Function ( HKDF)]]
  * RFC 5389 - [[http://tools.ietf.org/html/rfc5389|Session Traversal Utilities for NAT (STUN)]]
  * RFC 6238 - [[http://tools.ietf.org/html/rfc6238|TOTP: Time-Based One-Time Password Algorithm]]
    * RFC 4226 - [[http://tools.ietf.org/html/rfc4226|HOTP: Hash-Based One-Time Password Algorithm]]
  * RFC 6189 - [[http://tools.ietf.org/html/rfc6189|ZRTP: Media Path Key Agreement for Unicast Secure RTP]]
